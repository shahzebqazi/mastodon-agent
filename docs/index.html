Mastodon Fediverse Cursor Open source

What this is

This repository is a small, practical foundation for using Cursor (and your terminal) with the Mastodon API: compose posts as plain text, keep secrets in .env, and iterate with your agent without pasting tokens into chat.

The goal is to evolve it into a fuller posting and management toolkit as you use Mastodon more—drafts, scheduling ideas, media uploads, and admin helpers can layer on over time.

Quick start

  1. Clone the repo and copy .env.example to .env.
  2. In Mastodon: Preferences → Development → create an application with write:statuses (and add read scopes later if needed).
  3. Put your instance URL and access token in .env—never commit that file.
  4. Write your post in a UTF-8 file and run python3 scripts/post_status.py your_post.txt

Where it can go

  • Draft/review flow tuned for Cursor (templates, thread splitting)
  • Safe patterns for automation (rate limits, visibility defaults)
  • Optional media upload and alt-text helpers
  • Lightweight bookmarks or moderation notes for power users
