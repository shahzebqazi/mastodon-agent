mastodon-agent

Post and iterate on the fediverse from the same place you keep drafts and scripts—without putting tokens in chat or in git.

Mastodon Fediverse Python Open source

What this is

This repository is a small, practical foundation for posting to Mastodon via the API: compose posts as plain UTF-8 text, keep secrets in .env, and run python3 scripts/post_status.py … from your terminal or automation.

Product behavior and roadmap are defined in the tool-agnostic SPEC. Cursor-specific integration lives in the cursor-agents monorepo (mastodon-agent-cursor/).

Quick start

  1. Clone the repo and copy .env.example to .env.
  2. In Mastodon: Preferences → Development → create an application with write:statuses (and add read scopes later if needed).
  3. Put your instance URL and access token in .env—never commit that file.
  4. Write your post in a UTF-8 file and run python3 scripts/post_status.py your_post.txt

Where it can go

Phased roadmap (posting hardening → drafts → opt-in listen-log/curator): see the SPEC.
